## 🔧 Backend 更新

### 1. config.py
```python
DEVICE_GROUPS = {
    "全廠": ["全廠總用電"],
    "高壓盤": [6 個設備...],
    "洗砂設備": [2 個設備...],
    # ... 等等
}

ALL_DEVICES = []  # 自動生成 31 個設備列表
```

### 2. mock_data.py
為每個設備設定合理的基礎功率範圍：
```python
BASE_POWER_BY_DEVICE = {
    "全廠總用電": (800, 1500),
    "一號高壓盤總功率": (200, 400),
    "一號高壓盤總功率因數": (80, 95),    # 百分比
    "一號高壓盤總用電度": (500, 1000),   # kWh
    # ... 等等
}
```

### 3. Mock 數據特性
- **時間變化**：
  - 工作時間（8-17點）：+20% 負載
  - 夜間（0-5點）：-40% 負載
- **隨機波動**：±5% 高斯雜訊
- **周期波動**：60 秒正弦波
- **更新頻率**：每 10 秒一次

## 🆘 常見問題

**Q：為什麼設備數量從 18 變成 31？**
A：新版本添加了更詳細的設備指標，包括功率因數、用電度、各個子設備等，提供更細粒度的監控。

**Q：全廠總用電的計算方式是什麼？**
A：所有設備的即時功率值相加（包括水泵、風扇等輔助設備）。

**Q：可以隱藏某些設備嗎？**
A：目前不支援隱藏，但可以在 `config.py` 中修改 `DEVICE_GROUPS` 來自訂設備列表。

**Q：功率因數和用電度在哪裡？**
A：目前只生成功率值（kW）。功率因數（%）和用電度（kWh）已在 mock_data.py 中預留結構，等待真實 Power BI 數據源。
