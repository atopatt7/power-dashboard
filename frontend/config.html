Mock 模式說明

Mock 模式提供完全的測試環境，無需任何 Power BI 認證。系統會自動生成符合各設備群組特性的虛擬數據：

✓ 無需配置
直接啟動，無需任何認證密鑰
✓ 自動保存
所有數據保存至 SQLite
✓ 實時更新
每 10 秒自動生成新數據
✓ 時間模擬
工作時間負載增加 20%
🔐 Power BI 認證信息

填入你的 Azure/Power BI 認證信息。所有信息將加密保存到本地配置文件。

Azure AD 應用程式 ID
Azure AD 租戶 ID
Power BI 帳戶或服務帳號
帳戶密碼（ROPC 流程）
Power BI 資料集 ID
Power BI 工作區/群組 ID
📖 設置步驟
  1. 在 Azure AD 註冊應用程式
    • 前往 Azure Portal
    • 應用程式註冊 → 新增註冊
    • 複製應用程式 ID (Client ID) 和租戶 ID
  2. 獲取 Power BI 資料集信息
    • 開啟 Power BI Web
    • 選擇工作區 (Group ID 在 URL 中)
    • 選擇資料集，Dataset ID 在 URL 或設定中
  3. 設置服務帳號密碼
    • 建立專用服務帳號或使用現有帳號
    • 為 Power BI 資料集設定「編輯」權限
