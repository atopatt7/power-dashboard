## API 規格

### GET /api/power-latest
返回所有設備最新功率數據
```json
{
  "count": 18,
  "data": [
    {
      "timestamp": "2026-02-26T08:00:00.000000",
      "device_name": "一號高壓盤",
      "value": 450.5
    }
  ]
}
```

### GET /api/power-history
返回歷史功率數據，支援篩選
**參數：**
- `device` (optional): 設備名稱
- `start` (optional): 開始時間 (ISO format)
- `end` (optional): 結束時間 (ISO format)
- `limit` (default: 1000, max: 10000): 返回筆數

### GET /api/devices
返回設備群組配置
```json
{
  "groups": {
    "高壓盤": ["一號高壓盤", "二號高壓盤"],
    ...
  },
  "all_devices": [...]
}
```

### GET /api/health
系統健康檢查
```json
{
  "status": "ok",
  "mock_mode": true,
  "poll_interval": 10,
  "timestamp": "2026-02-26T08:00:00.000000"
}
```

## Mock 數據特性

- **隨機波動**: ±5% 的高斯雜訊
- **時間變化**: 工作時間 (8-17點) 負載 +20%，夜間 (0-5點) 負載 -40%
- **周期變化**: 60 秒週期的正弦波波動 (±10%)
- **更新頻率**: 每 10 秒一筆記錄
- **保留期**: 7 天
